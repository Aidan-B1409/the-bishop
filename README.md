# The Bishop bot

Joke Discord bot for a server I run. Also does basic role moderation.

## Setup

1. Copy `config.example.yml` to `config.yml`
2. Replace the placeholder token.
3. Add the desired role: roleID pairs.
4. `bundle install`
5. `bundle exec ruby main.rb`
